## 摘要

本研究結合傳統計量方法與機器學習，探討循環經濟實踐如何影響台灣的經濟發展與環境成果。以自行建構的 **22 個縣市、2014–2023 年 panel 資料**（220 筆觀察值，19 個變數），分析三個結果變數——家庭所得、人均用電量、住宅空屋率——對應各項循環經濟指標的關係。以 SHAP 值進行模型解釋與特徵重要性分析，結果顯示 **ML 模型預測力一致優於 panel 迴歸**，環境預算占比與再生能源裝置容量是最具影響力的特徵。

## 研究設計

### 應變數
| 變數 | 單位 | 說明 |
|---|---|---|
| 家庭可支配所得 | 千元新台幣 | 代表各行政區整體經濟產出 |
| 人均用電量 | 度/人 | 衡量地方層級能源使用效率 |
| 住宅空屋率 | % | 代理住房市場動態與地區經濟狀況 |

### 自變數（循環經濟指標）
- 一般廢棄物回收率（%）
- 環境保護預算結構比（%）
- 環境罰款占比（%）
- 人均焚化 / 掩埋廢棄物量（公斤）
- 地方政府人均資源回收量（公斤）
- 人均一般廢棄物產生量（公斤）
- 人均再生能源裝置容量（MW）
- 人均廚餘量（公斤）

### 資料來源
- **經濟與能源資料**：台灣電力公司
- **環境政策與污染資料**：行政院主計總處（DGBAS）

## 方法

### 線性基準：Panel 資料迴歸
- 固定效果模型（FE）與隨機效果模型（RE）
- Hausman 檢定選擇模型規格
- 主要結果以 FE 模型為主

### 非線性 ML 模型
| 模型 | 說明 |
|---|---|
| XGBoost | 序列提升，L1/L2 正則化 |
| LightGBM | Leaf-wise 樹生長，記憶體效率高 |
| CatBoost | Ordered boosting，適合類別變數 |
| Random Forest | Bagging 集成，並行樹構建 |
| SVM | 已排除——R² 為負，與本資料集相容性差 |

所有非線性模型均使用 **10-fold 交叉驗證**，以處理小樣本的過擬合問題。

### 可解釋性：SHAP
對全部四個集成模型計算 SHAP（SHapley Additive exPlanations）值，評估各特徵對預測結果的影響方向與大小。

---

## 主要結果

### 模型表現（R²）

| 結果變數 | XGBoost | Random Forest | LightGBM | CatBoost | FE 迴歸 |
|---|---|---|---|---|---|
| 家庭所得 | 0.720 | 0.688 | 0.699 | **0.752** | 0.578 |
| 人均用電量 | 0.427 | **0.479** | 0.474 | 0.420 | 0.054 |
| 空屋率 | 0.736 | 0.659 | 0.600 | **0.743** | 0.380 |

- **CatBoost** 在家庭所得與空屋率預測表現最佳；FE 迴歸明顯較弱，尤其在用電量預測上（R²=0.054）
- 人均用電量在所有模型中最難預測（R² < 0.5），可能受未觀測的行為與基礎設施因素影響

### 重要特徵（SHAP）
- **環境預算結構比**：在所有模型中對家庭所得與用電量均有最穩定的正向影響
- **人均廚餘量**：對家庭所得有強烈負向影響，對用電量正向影響——與初始假設相反
- **人均再生能源裝置容量**：對所得略有正向影響，但對用電效率的效果不一致

### 假設驗證結果
| 假設 | 內容 | 結論 |
|---|---|---|
| H1 | 環境預算與回收努力 → 提升用電效率 | **反向支持** |
| H2 | 高掩埋/焚化率 → 降低家庭所得 | 部分支持 |
| H3 | 廚餘與一般廢棄物 → 正向連結所得與用電 | 部分支持 |
| H4 | 再生能源裝置容量 → 提升所得與用電效率 | 部分支持 |
